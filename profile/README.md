Built into every tier of the platform, powered by Cloudflare Workers AI (Llama 3.1) with Vectorize for knowledge-base retrieval:

## 🧩 Technical Architecture

- **Multi-database design** — a dedicated CRM/application database, a multi-tenant database layer, a separate blog database, and MongoDB for NoSQL event/telemetry logging.
- **Real-time telemetry** via Firebase (public project reference: `webtechdomains-dd36f`) — logs events, page URL, user agent, geolocation, and IP info.
- **Geolocation** resolved via the OpenCage Geocoder API (used for lead capture and visitor analytics).
- **Currency conversion** via ApiLayer.
- **Timezone:** Asia/Kolkata (IST).

## 📦 Open Source & Internal Packages

We build and maintain our own PHP libraries that power the platform:

- **TaskScheduler** — cron-driven job/worker/queue scheduler with process management and structured logging.
- **Mysqldump** — pure-PHP MySQL backup/dump utility.
- **MultipleDB** — dynamic multi-tenant database connection manager for CodeIgniter.
- **Signer** — digital signing/verification library (powers e-signatures on contracts, estimates, and proposals).
- **Background_Process** — background task queue and execution engine with reporting.
